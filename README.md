# URL-Shortener-MS-With-Spring-Boot
URL Shortener service with Spring Boot.

## What is URL Shortener
An URL shortener is a website that reduces the length of your URL. The idea is to minimize the web page address into something thatâ€™s easier to remember and track. 

## Technologies used in this project.
1. Spring Boot
2. Base62 Encoding
3. Jackson JSON
4. JUnit - Mockito
5. Lombok
6. Slf4j

## How to Run?
You can create a docker image using Dockerfile and run the application as container.

After running the application, you can go to the Swagger and try APIs.
